# Claude Code Instructions

## Repository Policy

**Always create a clean, purpose-specific repository for each new app or tool built.**

- Do NOT build new features or apps on top of this `apex-capital-dashboard` repo.
- When starting a new project, create (or have the user create) a fresh repo named after exactly what is being built (e.g. `ai-tool-tracker`, `gov-bid-tracker`).
- Each repo should contain only the files for that one app — no mixing of unrelated projects.

## This Repo

This repo is for the **Apex Capital investment dashboard** only (`apex-dashboard.html`).
